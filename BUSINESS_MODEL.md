# MX Error Guide - Business Model

## Open Core

MX Error Guide follows the **Open Core** model, similar to GitLab and Elasticsearch.

### What's Open Source (MIT License)

| Component | Description |
|-----------|-------------|
| Error database | 376+ ISO 20022 error codes |
| Lookup UI | React search interface |
| Message guides | pacs.008, pacs.009, camt.053 |
| Scraper scripts | Data collection tools |
| E2E tests | Playwright test suite |

**Free forever**: Error lookup and basic features.

### What's Premium (Coming Soon)

Advanced validation features will be available as premium:
- Full XML validation
- Market practice rules
- Offline support
- Team features
